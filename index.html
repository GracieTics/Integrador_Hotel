Proyecto: SavoyConnect

Nombre del Proyecto:
SavoyConnect
Nombre de la Empresa:
Hotel Savoy
Problemática:
El Hotel Savoy maneja actualmente todas sus operaciones mediante un sistema obsoleto basado en hojas de papel y archivos Excel desorganizados, generando un caos operativo que afecta negativamente a los trabajadores, la experiencia del huésped y la coordinación entre departamentos y turnos.
Objetivo General:
Desarrollar un sistema de gestión (CRUD) con Python y SQL para reemplazar los métodos obsoletos. Este sistema centralizará la información para agilizar las operaciones diarias, mejorar la comunicación entre departamentos y turnos, optimizar el control de servicios para los huéspedes y facilitar el trabajo del personal.
